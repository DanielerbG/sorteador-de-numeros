<h1>🎲 Sorteador de números</h1>

<h2> 📖 Sobre</h2>
<p>Este projeto é uma aplicação web simples que permite sortear números aleatórios dentro de um intervalo definido pelo usuário.
O usuário pode informar quantos números deseja sortear e o intervalo mínimo e máximo para o sorteio.</p>

## ⚙️ Funcionalidades

- Definir a quantidade de números que serão sorteados.
- Definir o intervalo de números (valor inicial e valor final).
- Garantir que não haja números repetidos no sorteio.
- Validação de dados para evitar erros, como:
* O número inicial ser maior ou igual ao número final.
* Solicitar mais números do que o intervalo permite.
- Exibição dos números sorteados na tela.
- Botão Reiniciar para limpar os campos e realizar um novo sorteio.

## ▶️ Como executar o projeto

- Baixe ou clone este repositório.
- Abra a pasta do projeto no seu computador.
- Localize o arquivo index.html.
- Abra o arquivo em qualquer navegador (Google Chrome, Edge, Firefox, etc.).
- Preencha os campos com:
* Quantidade de números
* Número inicial
* Número final
- Clique no botão Sortear para gerar os números aleatórios.

O sorteio é feito utilizando a função Math.random() do JavaScript, garantindo números aleatórios dentro do intervalo definido pelo usuário.
